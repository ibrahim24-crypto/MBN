# **App Name**: MBN

## Core Features:

- Secure User Authentication: Implement Google Sign-in through Firebase Authentication, providing a streamlined and secure login process for all users.
- Role-Based Access Control: Establish and manage distinct user roles (Student, Teacher, Council, Administration) stored in Firestore to control access to specific features and content within the application.
- Super Admin Management Interface: Provide a dedicated dashboard for the super admin (ibrahimezzine09@gmail.com) to efficiently manage all user accounts, assign roles, and modify permissions.
- Student Profile & XP System: Automatically assign 100 XP to new student users upon successful sign-in, with XP values tracked and displayed as part of their user profile stored in Firestore.
- Council Announcement Board: Enable users with the 'council' role to publish and manage announcements or updates, displayed on a central board accessible to relevant user groups.

## Style Guidelines:

- Light color scheme, reflecting a professional and accessible educational environment. Primary color: a clean and trustworthy blue (#2499CC). This hue evokes stability and clarity.
- Background color: A very light, desaturated shade of the primary blue (#ECF5F9), providing a neutral and expansive canvas for content.
- Accent color: A soft blue-violet (#5C66CC), analogous to the primary, used for interactive elements, calls to action, and highlighting important information to maintain a cohesive yet distinct visual hierarchy.
- All text will use 'Inter', a modern grotesque sans-serif font known for its excellent legibility and clean appearance across various weights and sizes, ideal for both headlines and body text in an informational context.
- Utilize a set of modern, minimalist line icons to ensure clarity and improve navigation, providing visual cues for different roles and functionalities within the platform.
- Design a clean, structured, and fully responsive layout that adapts seamlessly to different devices. Emphasis will be on clear content hierarchy and intuitive navigation for all user roles, with distinct dashboard layouts for administrative users.
- Incorporate subtle, performance-optimized animations for interactive elements and state changes, providing gentle visual feedback to user actions and smooth transitions between pages.
